RA3 - Observabilidad, Seguridad y Ética

Este directorio organiza y documenta los entregables correspondientes a la
RA3 de la asignatura. Los scripts y archivos siguen en la raíz del proyecto;
estos README actúan como índice que mapea los requisitos del RA3 a los
archivos implementados.

Estructura de alto nivel:
- IL3.1 - Herramientas de Observabilidad: RA3/IL3.1/dashboard.py, RA3/IL3.1/config_logging.py
- IL3.2 - Trazabilidad y Logs: RA3/IL3.2/agente_metro.py, RA3/IL3.2/ejemplos.py, logs/
- IL3.3 - Seguridad y Ética: RA3/IL3.3/README.md y guardrails en RA3/IL3.2/agente_metro.py
- IL3.4 - Despliegue y Escalabilidad: RA3/IL3.4/Dockerfile, RA3/IL3.4/docker-compose.yml, scripts

Lee los README de cada IL para más detalles.
